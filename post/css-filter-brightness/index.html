Затемнение фона кнопки при наведении курсора мыши

September 25, 2020
CSS Tips & Tricks

Чтобы при наведении курсора сделать фон кнопки темнее, можно подобрать нужный цвет или буквально уменьшить яркость элемента с помощью CSS свойства filter: brightness()

button:hover {
  filter: brightness(0.94);
}





Иногда так получается быстрее, если заранее темные цвета не подобраны. Но стоит отметить, что белый текст становится серым и лучше слишком не затемнять.
