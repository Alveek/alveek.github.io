Игра: Array Puzzle

January 04, 2020
Игры

Поупражняйтесь в методах массива в JavaScript, решая мини-паззл.

Правила игры

Правила довольно простые. Да и игра сама займет минут пять на прохождение.

Нажав на кнопку НАЧАТЬ, на экране появится массив из перемешанных букв. В нём скрыт один из 5 популярных языков программирования. Требуется угадать, что это за язык и расставить буквы в правильном порядке.

Для этого используйте методы массивов:

  • pop() - извлечь последнюю букву
  • shift() - извлечь первую букву
  • push() - добавить букву в конец массива
  • unshift() - добавить букву в начало массива

Когда вы закончите расставлять буквы, необходимо преобразовать массив в строку с помощью метода join(""). Если вы угадали, то можно будет перейти к следующему слову нажав на кнопку ДАЛЕЕ, если нет, преобразуйте строку обратно в массив с помощью метода split("") и сделайте нужные изменения.
